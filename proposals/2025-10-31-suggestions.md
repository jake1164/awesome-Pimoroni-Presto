# Suggested additions for Presto awesome list (2025-10-31)

> Non-duplicates were attempted, but please double-check before merging. Each item includes a short description and the source link.

## Official docs & updates
- **Getting Started with Pimoroni Presto** — the canonical beginner guide covering launcher, examples, Wi‑Fi, and Thonny workflow. Source: https://learn.pimoroni.com/article/getting-started-with-presto
- **Presto firmware & example releases** — latest MicroPython builds (incl. "with-filesystem") on GitHub Releases. Source: https://github.com/pimoroni/presto/releases

## Tutorials & community projects
- **Presto Stream Deck Controller** — turn Presto into a Stream‑Deck‑style launcher; touch buttons mapped to hotkeys/macros. Code: https://github.com/goodeb/presto-stream-deck — Discussion: https://forums.pimoroni.com/t/stream-deck-style-controller-for-presto/26380
- **Last.fm Now Playing on Presto** — displays scrobbling info from Last.fm; clean UI and MicroPython examples. Code: https://github.com/andypiper/presto-lastfm
- **WiiM / "What’s Playing" display** — fetches currently playing track from a WiiM device and shows it on Presto. Discussion: https://forums.pimoroni.com/t/wiim-now-playing-on-presto/26123
- **MQTT Subscriber / Status Display** — lightweight MQTT client on Presto for dashboards and status. Discussion: https://forums.pimoroni.com/t/presto-a-mqtt-subscriber-device/26643
- **Flip Clock + 3D printable frame** — elegant flip clock with STL frame and build notes. Discussion: https://forums.pimoroni.com/t/retro-style-flip-clock-for-presto-with-3d-printed-frame/26791
- **Presto Maze** — interactive maze generator/solver with animated pathfinding and touch input. Code: https://github.com/kurosuke/PrestoMaze
- **Attitude Indicator for X‑Plane 12** — horizon/attitude instrument fed over UDP from the sim. Code: https://github.com/PaulskPt/Presto_attitude_indicator_for_X-Plane12
- **Solar System demo** — animated planets/orbits on Presto. Code: https://github.com/mrglennjones/presto-solar-system
- **Presto‑Cal (timezone & DST + iCal agenda)** — memory‑efficient TZ/DST helpers and iCal agenda app. Thread: https://forums.pimoroni.com/t/micropython-timezone-and-dst-library/28318 — Code: https://github.com/dan-parker/Presto-Cal
